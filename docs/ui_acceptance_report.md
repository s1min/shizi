## 二、全局设计令牌审计（uni.scss）

**文件**：`shizi-frontend/src/uni.scss`

这是最高优先级的整改项——全局设计令牌直接影响所有页面。

| 令牌 | v3 规范值 | 当前实际值 | 严重度 | 说明 |
|------|-----------|-----------|:------:|------|
| `$uni-color-primary` | `#F5A623`（暖杏色） | `#FFD700`（金色） | 🔴严重 | 使用了 v1 的高饱和金色，v2 已明确指出 #FFD700 饱和度过高易视觉疲劳 |
| `$uni-color-secondary` | `#5DADE2`（天空蓝） | `#87CEEB`（浅天蓝） | 🔴严重 | v1 色值，对比度不足 |
| `$uni-bg-color` | `#FFF8F0`（奶油白） | `#F0F2F5`（冷灰） | 🔴严重 | 完全偏离暖色调方向，当前为冷灰色调，与儿童产品温暖定位矛盾 |
| `$uni-text-color` | `#4A3728`（深棕） | `#333333`（纯深灰） | 🟡一般 | 缺少暖色调文字色，整体偏冷 |
| `$uni-color-success` | `#82C785`（薄荷绿） | `#52c41a`（Ant Design 绿） | 🟡一般 | 饱和度偏高，与 v3 低饱和护眼原则不符 |
| 字号单位 | rpx | px | 🔵建议 | 全局字号使用 px 而非 rpx，在不同屏幕密度下可能不一致 |

### 整改方案

```scss
// uni.scss 需要修改的值
$uni-color-primary: #F5A623;      // 暖杏色（v3）
$uni-color-secondary: #5DADE2;    // 天空蓝（v3）
$uni-bg-color: #FFF8F0;           // 奶油白（v3）
$uni-text-color: #4A3728;         // 深棕（v3）
$uni-color-success: #82C785;      // 薄荷绿（v3）
```
